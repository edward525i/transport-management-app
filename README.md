# Transport Management App

Aplicație desktop realizată în Python pentru gestionarea unei firme de transport.  
Folosește Tkinter pentru interfață și MySQL pentru baza de date.

## Funcționalități

- Autentificare
- Gestionare camioane, șoferi, transporturi, revizii și alocări
- Operații CRUD
- Interogări SQL (JOIN, subinterogări)
- Trigger pentru validarea alocării șoferilor
- Funcție și procedură stocată în MySQL

## Tehnologii

- Python 3
- Tkinter
- MySQL
- mysql-connector-python
- Pillow

## Instalare

Clonează repository-ul:

```bash
git clone https://github.com/edward525i/transport-management-app.git
cd transport-management-app
```
Instalează dependențele:
pip install mysql-connector-python pillow



Deschide MySQL Workbench

Rulează fișierul schema.sql

Verifică datele de conectare din transport_app.py


Rulare Aplicatie :
python transport_app.py

user: admin

parola: Unire
